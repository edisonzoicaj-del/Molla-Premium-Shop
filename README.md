# Molla Shop V3 Premium Static Ecommerce

Static Vercel-safe website. No npm, no build, no dependencies.

## Deploy settings
- Framework Preset: Other
- Install Command: empty
- Build Command: empty
- Output Directory: empty or ./
- Root Directory: ./

## Edit WhatsApp
Open script.js and change WHATSAPP_NUMBER.

## Upload to GitHub
Upload the CONTENT of this folder: index.html, styles.css, script.js, vercel.json, README.md, assets/.
